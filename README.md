# Kuiz Vokal AEIOU 3D

Kuiz interaktif 3D Vokal AEIOU untuk murid prasekolah.

## Fungsi

- Soalan vokal Bahasa Melayu AEIOU
- Objek animasi gaya 3D
- Audio menggunakan Web Speech API
- Skor automatik
- Paparan mesra kanak-kanak
- Boleh dibuka melalui pautan web

## Teknologi

- HTML
- CSS
- JavaScript
- GitHub Pages

## Nota

Versi awal ini menggunakan emoji animasi gaya 3D sebagai placeholder. Model 3D sebenar (.glb) boleh ditambah kemudian dalam folder `models/`.
